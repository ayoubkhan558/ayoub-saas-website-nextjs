# Project structure

The project now separates app routes, reusable UI, content, features, and utilities.

```txt
src/
  app/                 Next.js route files and route metadata
  components/          Reusable shared/layout UI components
  content/             Editable portfolio, work, and tool data
    portfolio/         Global profile, homepage, services, testimonials
    work/              Projects, case studies, archive, link status
    tools/             Tool/logo catalog
  context/             React context providers
  data/                TypeScript adapters for content imports
  lib/                 SEO, pagination, and shared utilities
  styles/              Global design tokens, mixins, animations
```

## Positioning

Primary focus: custom React.js, HTML, CSS, JavaScript, Next.js, conversion-focused front-end builds, Figma-to-code, and landing pages.

Secondary focus: WordPress, WooCommerce, and CMS work when the client needs editable content or a practical handoff.
